# Run Template

This tracked tree is the neutral starter workspace for a new run.

It MUST contain only reusable placeholders and starter directory shape.

It MUST NOT contain:

- a real domain brief
- accepted role output from a prior run
- completed evidence from a prior run
- fixed capability decisions for a specific product

It SHOULD contain:

- neutral starter gating files such as `artifacts/architecture/capability-profile.md`
  and `artifacts/architecture/load-plan.md`
- starter directory shape for optional run-owned lanes that may be activated

Create local `runs/current/` by copying this tree before a new run starts.
