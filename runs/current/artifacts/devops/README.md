# DevOps Artifacts

This directory is for optional run-owned DevOps artifacts.

Use it only when the DevOps role is active for the run.

These artifacts capture:

- package-management policy for the generated app
- packaging and same-origin delivery decisions
- build and runtime verification commands
- packaging verification evidence and packaged acceptance notes

Required files when the role is active:

- `package-policy.md`
- `packaging-plan.md`
- `build-matrix.md`
- `verification.md`
