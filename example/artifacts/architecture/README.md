# Preserved Architecture Example: Cimage

This directory contains a preserved filled architecture package for the
example Cimage app.

It exists to show the expected shape of completed architecture artifacts.

Rules:

- These files are example reference material only.
- Fresh runs MUST write their own architecture artifacts under
  `../../runs/current/artifacts/architecture/`.
- Agents MAY use this directory as a formatting and completeness reference.
- Agents MUST NOT copy these decisions blindly into unrelated domains.
