# Escalation

Escalate to the Architect immediately when:

- frontend and backend disagree on resource naming
- `admin.yaml` and the starter domain diverge
- docs rely on hidden runtime files not included in this playbook
- tests pass only because of assumptions absent from the docs
- product language and backend rule implementation diverge
- custom pages require unsupported query/runtime behavior

Escalate to the Product Manager immediately when:

- requested behavior is out of scope
- a derived field turns out to be a business-critical input
- a UI tweak changes workflow semantics
- two user groups require conflicting landing pages or layouts
