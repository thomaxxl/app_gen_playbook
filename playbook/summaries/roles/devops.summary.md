# DevOps Role Summary

Use this role for advanced packaging ownership: runtime/toolchain packaging,
same-origin delivery, Docker/Compose refinement, nginx, and packaging
verification.

Always load:

- `global-core.md`
- `process-core.md`
- `../../process/read-sets/devops-core.md`

This role does not own product behavior, UX decisions, backend semantics, or
business-rule logic. Baseline packaging still exists even when this role is
inactive.

Load frontend/backend packaging details only when the current task needs them.

Full docs:

- `../../roles/devops.md`
- `../../../specs/contracts/deployment/README.md`
