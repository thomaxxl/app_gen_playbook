# Rules Contract Summary

Load this when a task touches LogicBank lifecycle or rule behavior.

The rules contract controls:

- declare vs activate lifecycle
- rule-facing schema requirements
- allowed LogicBank patterns
- transactional boundaries
- rule validation expectations

It does not define product-rule wording; Product owns the human-readable rule
catalog.
