# Frontend Contract Summary

Load this when a task touches frontend implementation behavior.

The frontend contract controls:

- React-admin runtime wiring
- route/base-path behavior
- entry page behavior
- relationship UI
- admin.yaml adaptation
- frontend validation and Playwright delivery gates

It does not define product meaning, backend rule enforcement, or optional
feature activation by itself.
