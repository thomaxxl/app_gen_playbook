# dnd-kit Activation

Enable this pack only when the run explicitly needs:

- drag/drop placement
- sortable or reorderable lists
- Kanban-like boards
- repositionable UI elements

Install pins:

- `@dnd-kit/react@0.3.2`
- `@dnd-kit/helpers@0.3.2`

New work MUST prefer these packages over the older `@dnd-kit/core` line.
