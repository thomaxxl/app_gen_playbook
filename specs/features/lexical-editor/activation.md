# Lexical Editor Activation

Enable this pack only when the run explicitly needs:

- rich-text authoring
- formatted notes or descriptions
- structured content blocks
- markdown or HTML import/export around editable content

Base install profile:

- `lexical@0.41.0`
- `@lexical/react@0.41.0`
- `@lexical/rich-text@0.41.0`
- `@lexical/history@0.41.0`
- `@lexical/link@0.41.0`
- `@lexical/list@0.41.0`
- `@lexical/utils@0.41.0`

All Lexical packages in a run MUST stay on the exact same version line.
