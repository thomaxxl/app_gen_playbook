# Embla Carousel Activation

Enable this pack only when the run explicitly needs:

- image sliders
- testimonial or proof carousels
- horizontal card decks that must swipe on mobile

Install pins:

- `embla-carousel-react@8.6.0`
- optional only when autoplay is explicitly requested:
  - `embla-carousel-autoplay@8.6.0`

This pack SHOULD remain exceptional in admin apps.
